Bootstrap SquizNav

This script moves items nav in dropdown if the container is not large enough.

To see the magic happening, resize your window!

Using HTML

<ul class="nav nav-tabs" data-squiznav="">

 

Using Javascript

$('.squiz-me').squizNav();
